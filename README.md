## Bitka bendov — Multi-step prijavni obrazec

**Kratek opis**
Napreden večkorakni (multi-step) obrazec za prijavo glasbenih izvajalcev, z dvojezično podporo, validacijo, prijaznimi prehodi in lepimi popup obvestili (SweetAlert2). Namen: festival, natečaj ali battle-of-the-bands prijave.

## Clash of the Bands — Multi-Step Registration Form

**Brief description**  
A polished multi-step registration form for musicians/bands featuring bilingual support, validation, smooth transitions and SweetAlert2 notifications. Ideal for festivals, contests and battle-of-the-bands events.
